\# 智能投资监控系统



一个功能完整的股票监控系统，支持自选股管理、实时监控、筹码分析、钉钉提醒等功能。



\## 功能特点



\- 📊 \*\*自选股管理\*\*：添加/删除自选股，实时查看行情

\- 📈 \*\*实时监控\*\*：监控单个股票的实时价格、涨跌幅

\- 🎯 \*\*筹码分析\*\*：分析股票筹码集中度，判断买卖时机

\- ⚠️ \*\*涨跌提醒\*\*：设置涨跌幅阈值，钉钉自动提醒

\- 📝 \*\*规则设置\*\*：自定义买卖规则，自动触发信号

\- 🔔 \*\*钉钉集成\*\*：支持钉钉机器人消息推送

\- 💼 \*\*持仓管理\*\*：记录成本价，计算实时盈亏
